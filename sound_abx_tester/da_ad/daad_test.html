File for Verifying Sound Quality Degradation through DA/AD Conversion and Listening Test

This is a file and listening test collection to verify the degree of sound quality degradation caused by DA/AD conversion, created using Focusrite Scarlett 18i20 Generation 4.

Conditions for File Creation

  • Connected LineOut 7/8 to Line In 7/8 of Focusrite Scarlett 18i20 Generation 4 using 1m unbalanced cables
  • Set the gain of Line In 7/8 to 0dB and the playback volume of the music file to -12dB (adjusted to minimize distortion during measurement)
  • Played back the original 96kHz/24bit file and recorded simultaneously
  • Normalized the loudness of both the original and recorded audio to align their volumes

DA/AD変換による音質劣化確認用ファイルと聞き分けテスト

Focusrite Scarlett 18i20 Generation 4を用いて作成した、DA/AD変換による音質劣化度合いを確認するためのファイルと聞き分けテスト問題集です。

【ファイル作成時の条件】

  • Focusrite Scarlett 18i20 Generation 4のLineOut 7/8をLine In 7/8にそれぞれ1mのアンバランスケーブルで接続
  • Line In 7/8のGAINは0dB、音楽ファイルの再生音量を-12dBに設定（測定における歪が最も小さくなるよう調整した結果）
  • 96k24bitのオリジナルファイルを再生し、同時に録音
  • 音量を揃えるため、オリジナル・録音音声をそれぞれラウドネスでノーマライズ
